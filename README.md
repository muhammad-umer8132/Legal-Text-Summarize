## 🚀 Quick Start

### Prerequisites
- Python 3.8 or higher
- Node.js 16 or higher
- npm or yarn
- Google Chrome (for Playwright)

### Installation

#### 1. Clone the Repository
```bash
git clone <repository-url>
cd Legal_text_summarizer
```

#### 2. Backend Setup
```bash
cd backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows
.\venv\Scripts\Activate.ps1
# Mac/Linux
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Install Playwright browser
playwright install chromium

# Start the backend server
python main.py
```

#### 3. Frontend Setup
```bash
# Open new terminal
cd frontend

# Install dependencies
npm install

# Start the development server
npm start
```

#### 4. Access the Application
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8000
- **API Documentation**: http://localhost:8000/docs

## 📖 Usage

### Web Interface
1. Open http://localhost:3000 in your browser
2. Upload a legal document (PDF or TXT) using:
   - Click "Choose Your Document" button
   - Drag and drop files into the upload area
3. Click "Analyze Document" to start AI analysis
4. Monitor real-time progress updates
5. View the structured legal analysis results
6. Download the analysis as a formatted text file

For quick analysis without the web interface:

## 🎯 IRAC Analysis Method

The application analyzes legal documents using the IRAC method:

- **Issue**: Identifies the central legal questions
- **Rule**: Lists applicable laws and precedents
- **Analysis**: Applies rules to case facts
- **Conclusion**: States the final outcome
- **Legal Advice**: Provides plain English guidance

## 🔧 Troubleshooting

### Common Issues

#### 1. "ModuleNotFoundError: No module named 'playwright'"
```bash
# Activate virtual environment
.\venv\Scripts\Activate.ps1

# Install dependencies
pip install -r requirements.txt
playwright install chromium
```

#### 2. "Gemini login required"
- Log in to Google/Gemini in your browser first
- Ensure you're logged into the correct Google account
- Try accessing https://gemini.google.com manually

#### 3. Frontend not connecting to backend
- Ensure backend is running on port 8000
- Check CORS settings in backend
- Verify both services are running simultaneously

#### 4. File upload failing
- Check file format (PDF/TXT only)
- Ensure file size is under 10MB
- Verify file is not corrupted

#### 5. npm command not found
- Install Node.js from https://nodejs.org/
- Restart terminal after installation
- Verify installation: `node --version` and `npm --version`
